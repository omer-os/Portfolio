House builder

What it is

A piece of software that allowed the user to draw rooms and set properties such as names, interior carpet and painting prices before being given a price for full decoration in return. Created over the course of 3-4 days during after school hours using pygame.

What I did

Designed and created the systems visuals, logic and code over a limited time period. I designed the user interface with an aim to create an intuitive system that could be used with little to no tutorial and through testing with users feel that I succeeded at this.
